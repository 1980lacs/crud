Mi aplicación chula

La aplicación es un CRUD que sirve para:

  1. Inserción de empleados y proyectos
  2. Modificación de empleados y proyectos
  3. Asignacición de proyectos a empleados
  4. Asignacición de empleados a proyectos
  5. Eliminación de empleados, proyectos y asignaciones
  6. Listado de empleados y proyectos
  7. Busqueda de empleados y proyectos
  8. Crear la base de datos con datos de muestra

Para un futuro se podrá crear una base de datos vacía
